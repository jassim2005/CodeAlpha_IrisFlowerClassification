# 🚀 CodeAlpha Data Science Internship (June 2026)

## 📌 Project Title
**Machine Learning Projects: Iris Flower Classification, Car Price Prediction & Sales Prediction**

---

## 👨‍💻 About the Internship

This repository contains **three machine learning projects** completed as part of the **CodeAlpha Data Science Internship (June 2026)**.  

Each project focuses on real-world supervised learning problems including classification and regression using Python and popular data science libraries.

---

## 📂 Projects Overview

### 🌸 1. Iris Flower Classification (Classification Problem)

This project predicts the species of an Iris flower based on its physical features.

**Features Used:**
- Sepal Length  
- Sepal Width  
- Petal Length  
- Petal Width  

**Target:**
- Setosa  
- Versicolor  
- Virginica  

**Algorithms Used:**
- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Decision Tree / Random Forest  

**Goal:**
To classify flower species accurately using supervised machine learning.

---

### 🚗 2. Car Price Prediction (Regression Problem)

This project predicts the selling price of used cars based on various features.

**Features Used:**
- Year  
- Present Price  
- Kms Driven  
- Fuel Type  
- Transmission  
- Seller Type  

**Algorithms Used:**
- Linear Regression  
- Random Forest Regressor  

**Goal:**
To estimate accurate car prices and understand factors affecting vehicle pricing.

---

### 📈 3. Sales Prediction (Regression Problem)

This project predicts future sales based on advertising or historical sales data.

**Features Used:**
- Advertising Spend (TV, Radio, Newspaper) OR historical sales features  
- Market trends  

**Algorithms Used:**
- Linear Regression  
- Regression Models (as per dataset)

**Goal:**
To analyze patterns and predict future sales performance.
